# Programimi-me-Book-Cipher-dhe-Route-Transposition-
Ky projekt implementon një server të thjeshtë në Java që është në gjendje të pranojë dhe të menaxhojë lidhje nga shume klientë njëkohësisht. Serveri përdor socket programming dhe multithreading për të siguruar komunikimin paralel me secilin klient.
